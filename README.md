# pymongo-api

## Как запустить

Перейти в директорию с проектом

```shell
cd sharding-repl-cache
```

Запускаем mongodb, redis и приложение

```shell
docker compose up -d
```

Заполняем mongodb данными

```shell
./mongo-init.sh
```

## Как проверить

Откройте в браузере http://localhost:8000
